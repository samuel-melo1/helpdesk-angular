Help Desk Dicas de Um Dev

Olá! Seja bem vindo ao sistema HelpDesk desenvolvido por mim no curso de Desenvolvedor Full Stack.
Esse é um sistema de gerenciamento de chamados simulando um simples HelpDesk desenvolvido em Angular e Spring Boot 2.
O sistema está rodando na nuvem sendo o front no Railway e o Backend também no Railway.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização, Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link do Github no menu lateral.
