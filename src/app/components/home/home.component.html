Help Desk Central de Chamados

Olá! Seja bem vindo ao sistema HelpDesk - Central de Chamados.
Esse é um sistema de gerenciamento de chamados simulando um simples HelpDesk desenvolvido em Angular e Spring Boot .
O sistema está rodando na nuvem sendo o front no Vercel e o Backend no Railway.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização, Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link do Github no menu lateral.
